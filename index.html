📘 Panduan Google Sheets
Langkah-langkah setup Google Sheets:
1. Buka Google Sheets dan buat spreadsheet baru
2. Buat 3 sheet: Siswa, Absensi, Konfigurasi
3. Sheet Siswa: kolom NIS, Nama, Kelas, Gender, NoWA
4. Sheet Absensi: kolom NIS, Nama, Kelas, Tanggal, Waktu, Status, Keterangan
5. Buka Extensions → Apps Script
6. Paste kode Apps Script (tekan tombol di bawah untuk salin)
7. Deploy → New Deployment → Web App → Execute as Me → Anyone
8. Salin URL dan masukkan di atas
